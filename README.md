# 1971 Capital — cloud skyline hero

A static, no-build React site (JSX transpiled in-browser via Babel standalone).
The skyline is a fixed "bookend" layer that appears at the hero and footer, an
animated clouds GIF backs the thesis scroll-words section, and section copy
rises out of / fades into the fog.

## Run locally
```bash
python3 -m http.server 5171
# open http://localhost:5171/
```

No build step — edit the `.jsx` / `.css` files directly.
